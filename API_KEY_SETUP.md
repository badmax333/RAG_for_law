# Настройка API ключа Mistral AI

## Шаг 1: Получите API ключ

1. Зарегистрируйтесь на https://console.mistral.ai/
2. Перейдите в раздел API Keys
3. Создайте новый API ключ
4. Скопируйте ключ

## Шаг 2: Настройте ключ в проекте

### Способ 1: Создайте файл .env (рекомендуется)

1. Создайте файл `.env` в корне проекта (рядом с `main.py`)
2. Добавьте следующую строку:
```
MISTRAL_API_KEY=ваш_ключ_здесь
```

Например:
```
MISTRAL_API_KEY=mistral-abc123xyz789...
```

### Способ 2: Отредактируйте config.py

Откройте файл `config.py` и замените строку:
```python
MISTRAL_API_KEY = "your_mistral_api_key_here"
```

на:
```python
MISTRAL_API_KEY = "ваш_реальный_ключ"
```

## Шаг 3: Проверка

После настройки ключа запустите:
```bash
python main.py "Тестовый вопрос"
```

Если все настроено правильно, система подключится к API и начнет работу.

## Важно!

- ⚠️ НЕ коммитьте файл `.env` в git (он уже в .gitignore)
- ⚠️ НЕ публикуйте ваш API ключ
- ⚠️ API вызовы платные - следите за использованием на https://console.mistral.ai/
